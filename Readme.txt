Working: 
- Uses the Output.txt from the lexer to parse the tokens in it one by one.
- So run the lexer first to make the Output.txt file. Then run the Parser.
- On successful run will generate 3 files: tac.txt, parsetree.txt and symboltable.txt.
- On unsuccessful run will print the error.
